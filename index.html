Ocam by toddlucas

Razor and Markdown static site generator

Ocam is a static site generator in the spirit of Jekyl that can be used to build blogs and other content rich sites, without the need for any database or server side runtime capabilities. It is written in C#, runs on the .NET platform, and uses the RazorEngine. This means that those familiar with ASP.NET MVC can apply similar techniques to bulding static sites as they are accustomed to with the MVC platform.
